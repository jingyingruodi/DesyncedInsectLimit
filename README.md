# Desynced 虫群单位上限修改模组 (Insect Limit Mod)

这是一个专为提升多人服务器体验而设计的逻辑修改模组（Addon）。

## 📖 模组背景 (Background)
在《Desynced》原版逻辑中，全图虫群单位数量有硬性上限（4,000只），且当总数超过 2,000 只时，刷怪速率会显著降低。
在单人模式下这足以保证虫群活力，但在多人服务器中，由于多名玩家共用这一上限，导致“人均虫子”数量极低，严重影响了防守玩法的乐趣。

## ✨ 核心功能 (Features)
本模组通过重写虫群生成器逻辑，大幅提升了上限，恢复了多人模式下的防守强度：
- **绝对上限提升**：将虫群最大单位数从 **4,000** 提升至 **30,000**。
- **衰减闸值上限提升**：将生成数量开始衰减的阈值从 **2,000** 提升至 **20,000**。
- **扩张闸值上限提升**：虫群派遣侦察兵建立新巢穴的判定阈值同步提升至 **20,000**。
- **完美兼容专用服务器**：纯逻辑修改，无任何 UI 组件，适用于大规模专用服务器环境。

## 🛠 安装方法 (Installation)
1. 下载本项目文件。
2. 将项目文件夹放入游戏的 `Content/mods/` 目录下。
3. 启动游戏并在模组菜单中启用。
4. **服务器部署**：直接将文件夹放入服务器的 `mods` 目录，并在配置文件中确保其被加载。

## ⚠️ 注意事项
- 提升至 30,000 单位会对 CPU 造成较大压力，请根据服务器配置酌情在 `data.lua` 中调整数值。
- 模组仅修改单位上限，不改变虫子单体的数值（血量、攻击力）。
