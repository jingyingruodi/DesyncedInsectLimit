# Desynced 虫群单位上限修改模组 (Insect Limit Mod)

这是一个专为提升多人服务器体验而设计的逻辑修改模组（Addon）。

## 📖 模组背景 (Background)
在《Desynced》原版逻辑中，全图虫群单位数量有硬性上限（4,000只），且当总数超过 2,000 只时，刷怪速率会显著降低。
在单人模式下这足以保证虫群活力，但在多人服务器中，由于多名玩家共用这一上限，导致“人均虫子”数量极低，甚至因为其他区域积压虫子导致玩家附近不刷怪，严重影响了多人防守玩法的乐趣。

## ✨ 核心功能 (Features)
本模组通过精准重写虫群生成器与扩张 AI 逻辑，在维持原版难度的基础上大幅提升了多人承载力：
- **绝对上限提升**：将虫群最大单位数从 **4,000** 提升至 **30,000**，确保多人分摊后依然有怪可打。
- **生成波次优化**：将生成数量开始缩减的阈值从 **2,000** 优化至 **4,000**，平衡性能与进攻烈度。
- **扩张范围放宽**：虫群派遣侦察兵建立新巢穴的判定阈值提升至 **10,000**，允许虫群在更广阔的地图上分布。
- **智能防扎堆 AI**：引入聚落限额控制（35格半径内最多允许3个巢穴），防止多名侦察兵在同一地点密集筑巢导致性能崩溃。
- **完美“保原味”逻辑**：完整保留并恢复了原版的距离难度加成、高地判定及视觉特效等核心机制。
- **修复官方原版 Bug**：解决了虫群会锁定并无限围攻“建筑规划（蓝图）”导致单位堆积的顽疾。
- **极致健壮性优化**：修复了原版 AI 在母巢意外销毁或资源搜索过程中的逻辑崩溃错误。
- **服务器完美兼容**：纯逻辑修改，无 UI 组件，完全支持专用服务器（Dedicated Server）及自动同步。

## 🛠 安装方法 (Installation)
1. 下载本项目文件。
2. 将项目文件夹（或打包好的 `InsectLimit.zip`）放入游戏的 `Content/mods/` 目录下。
3. 启动游戏并在模组菜单中启用。
4. **服务器部署**：直接将 ZIP 包放入服务器的 `mods` 目录即可。

## ⚠️ 注意事项
- 30,000 单位是理论上限，实际运行压力取决于服务器 CPU 性能。
- 模组仅修改单位上限与分布逻辑，不改变虫子单体的属性（血量、攻击力）。
