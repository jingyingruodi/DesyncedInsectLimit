# Desynced 虫群单位上限修改模组 (Insect Limit Mod)

这是一个专为修复原版 AI 缺陷、优化生命周期并引入活跃玩家动态权重的虫群增强模组。

## 📖 版本特性 (v2.7.20 Aligned Final)

### 1. 行为解耦与原版逻辑对齐 (New!)
- **三路独立冷却**：派遣、进攻与扩张行为的全局冷却完全分离，不同蜂巢可并行决策，大幅提升多人环境下的 AI 灵活性。
- **多人动态权重**：频率限制随**存活玩家**缩放（保护 UPS），单位上限按**总玩家数**扩展（维持世界强度）。

### 2. 动态可扩张上限系统
- **动态绝对上限**：基础 12,000 只战斗单位，每增加一名玩家（无论是否在线）额外 +3,000。
- **动态削弱阈值**：基础 4,000 只，随活跃玩家数动态增长，超过后刷怪量减半，维持服务器性能。
- **实时监控**：Log 后台每 30 秒播报 `Players: 存活/总数 | BOTS: 当前/上限`。

### 3. 智能扩张与导航修复
- **前线单位锁定**：侦察虫动态锁定物理距离最近的玩家单位，有效克制“移动基地”规避战术。
- **超视距导航**：解锁寻路限制，支持 700+ 格超远距离渗透，并具备 180 秒（900 ticks）卡死处决容错。
- **归巢死循环修复**：当归巢路径被地形阻断时，该巢穴会被拉黑 250 秒，强制虫子寻找其他出路。

### 4. 生命周期与性能优化
- **精准战斗判定**：重构卡死检测。只有在真正射击或受损时才重置卡死计时，彻底解决“残血发呆”单位永久占位的问题。
- **目标合法性校验**：底层过滤蓝图、矿点、中立遗迹，彻底解决单位在无效目标点堆积的 Bug。
- **30s 极效普查**：采用 150 ticks 低频缓存统计，确保大规模单位下的性能开销降至 O(1)。

## 🛠 安装方法
1. 将模组文件夹放入游戏的 `Content/mods/` 目录下。
2. 在模组菜单中启用 `InsectLimit`。
